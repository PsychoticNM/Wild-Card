# <p align="center"> 1.0.0.1 - December 19th, 2025 </p>

**Notes:**
- This hotfix contains various mod updates, some profile changes, and important fixes for playability
<br></br>

**Updating:**
- Please refer to [Updating Nuclear Sunset](<https://nuclearsunset.com/docs/main/updating>) for further instructions.
- Altered or outdated lists will not be supported.
- Notifications of missing mods post-update are normal and reflect plugin merging.
<br></br>

**Testing:**
- Please request the test role in the [mod-testing](https://discord.com/channels/1445447774035251323/1445462359681532155) Discord channel under the `Nuclear Sunset Tester - Role Requesting Thread` thread.
- This role will be stripped from those who are non-participatory, provide low quality feedback, ask inappropriate installation questions, or otherwise abuse the role.
<br></br>

 **Community:**
- If you’re enjoying the NS experience, please endorse the [Nuclear Sunset Nexus](<https://www.nexusmods.com/newvegas/mods/94967>) page, which helps with visibility in Nexus Collections.
- Also please consider leaving a review in the comments on the [Nuclear Sunset Nexus](<https://www.nexusmods.com/newvegas/mods/94967?tab=posts>) page or on reddit! We value your feedback!
- Submit Nuclear Sunset screenshots to the [ns-screenshots](https://discord.com/channels/1445447774035251323/1445472863455805684) Discord channel for a chance to be featured on the website or loading screens!
- Submit scenarios for Immersive Fast Travel or Immersive Sleep in the [submissions](https://discord.com/channels/1445447774035251323/1445462646424993976) Discord channel!
- Submit custom Character Builds using the [Nuclear Sunset Build Planner](https://mrgassy.github.io/Nuclear-Sunset-Build-Planner/) to the [character-builds](https://discord.com/channels/1445447774035251323/1445462178172895262) Discord channel!
<br></br>

**Fixes:**
- Fixed FPS issues due to scripts spamming the console.
- Fixed 21:9 MAPMO preset
- Fixed conflict between Forced Entry and Dynamic Interaction Prompts
- Fixed default keybinds for new installs
- Fixed text not correctly referring to Nuclear Sunset in warning when opening Tweaks menu
- Fixed Dialogue History not persisting when updating the list
- Fixed missing tree assets from Zion Trail Redux
- Fixed tutorial message saying NuclearSunset.net instead of NuclearSunset.com
<br></br>


**Changes:**
